Nigerian Army InfoCenter - Site Plan

Site Name

Nigerian Army InfoCenter

This name was chosen because it clearly represents a central hub where users can find reliable information and news about the Nigerian Army.

Optional domain: nigerianarmyinfocenter.org

Site Purpose

The website will provide general information and news about the Nigerian Army. It will include history, roles, latest updates, and public information to help educate citizens and improve awareness.

Scenarios

  • Where can I find the latest Nigerian Army news updates?
  • What are the responsibilities of the Nigerian Army?
  • How can I contact the Nigerian Army for official inquiries?

Color Schema

Army Green (#2E4F2E) – Used for header and headings

Gold (#C9A227) – Used for highlights and accents

Light Gray (#F4F4F4) – Used for page background

Typography

Montserrat – Used for headings

Roboto – Used for body text
